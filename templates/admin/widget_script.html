Widget Embed Script

Copy the snippet below and paste it into the <body> of any webpage to embed the Gujarat Vidyapith AI chatbot bubble.

Minimal (uses all defaults)

<script
  src="{{ server_url }}/static/js/widget.js"
  data-server="{{ server_url }}"
></script>

Full (all customisation options)

<script
  src="{{ server_url }}/static/js/widget.js"
  data-server="{{ server_url }}"
  data-position="bottom-right"
  data-color="#735557"
  data-title="Ask our AI"
  data-greeting="Hello! How can I help?"
  data-mode="popup"
></script>

Attribute reference

Attribute Default Description
data-server required Base URL of this Gujarat Vidyapith AI server — no trailing slash.
data-position bottom-right bottom-right or bottom-left — where the bubble appears.
data-color #735557 Accent colour for the bubble and widget header (any CSS colour value).
data-title Gujarat Vidyapith AI Title shown in the widget header.
data-greeting none Opening message shown as the first assistant bubble when the widget opens.
data-mode popup popup — compact corner popup. fullpage — full viewport overlay.

Preview the widget directly

Tip: The data-server value is auto-filled with this server's URL ({{ server_url }}). Update it if you deploy to a different domain.
